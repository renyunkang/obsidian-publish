# 重启网络

  • ifup -v eth0 / ifdown eth0
  • ifconfig eth0 up/down
  • service network restart 或者/etc/init.d/networking restart

添加默认路由

  • route -A inet6 add ::/0 gw 2001:da8:203:ec7::1

添加路由

  • route add default gw 192.100.10.1
  • ip route add default via 192.168.0.1
  • ip route add 192.100.10.0/24 via 192.100.200.1 dev enp49s0f1
  • route add -net 192.100.10.0 netmask 255.255.255.0 dev eth0

# vlan 配置

vlan 配置

1、  加载8021q模块

  • lsmod |grep ‘8021q’ 查看模块加载成功与否
  • modprobe  8021q  命令加载内核模块
  • 没有的话安装 apt-get install vlan
  1. 创建子接口
  • ip link add link eth1 name eth1.10 type vlan id 10
  • ip addr add 10.10.10.2/24 dev eth1.10
  • ip link set dev eth1.10 up
